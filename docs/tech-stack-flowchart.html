How It Works
The 8-Step Analysis Pipeline
Every analysis runs these stages in sequence, with parallel execution where possible to maximize speed.
1
URL Submission & Request Routing
A user submits their website URL. The request is received by a serverless function configured for up to 300 seconds of compute time, ensuring even slow-loading sites are fully analyzed. A rate-limit check fires immediately to prevent abuse.
Vercel Upstash Redis
2
Web Scraping & Content Extraction ▲ Parallel
The site's full HTML and CSS are fetched. For bot-protected sites (Cloudflare, etc.), Firecrawl bypasses restrictions and returns clean, rendered content. Both inline styles and external stylesheets are collected. Runs simultaneously with screenshot capture.
Firecrawl Native Fetch
3
Screenshot Capture ▲ Parallel
A full 1440×900 desktop screenshot of the live site is taken via the ScreenshotOne API — with ads and cookie banners automatically blocked to show the real page content. Runs simultaneously with web scraping.
ScreenshotOne
4
AI Visual & SEO Analysis ▲ Parallel
Three tasks run simultaneously: (1) Claude Vision analyzes the screenshot for visual quality, layout hierarchy, trust signals, CTA placement, and messaging clarity. (2) Claude AI reads the HTML to detect industry, extract headline copy, and audit SEO metadata. (3) Brand assets — colors, fonts, images, and logo candidates — are extracted and uploaded to S3.
Claude Vision (Haiku) Claude AI (Haiku) AWS S3 AWS RDS
5
Logo Identification ▲ Parallel
Claude Vision reviews up to 5 logo candidates extracted from the page and identifies the true brand logo with a confidence score. This ensures the correct logo — not a random icon — appears in all three generated designs. Runs in parallel with scoring.
Claude Vision (Haiku)
6
Scoring Against Industry Benchmarks ▲ Parallel
Claude AI scores the site across 8 dimensions — Clarity, Visual Quality, Layout Hierarchy, Trust Signals, Conversion, Content, Mobile, and Performance — then compares each score to real industry benchmarks stored in the database to produce a competitive percentile ranking.
Claude AI (Sonnet) AWS RDS
7
AI Design Generation — 3 Layouts
Three specialized Claude AI creative agents each generate a complete, production-ready HTML landing page redesign — Modern, Classy, and Unique — using the site's real branding, images, copy, and confirmed logo. Each agent launches with a short stagger to maximize throughput. Results are saved to the database as each layout completes.
Claude AI (Sonnet) ×3 AWS RDS
8
Results Delivery & Storage
The original screenshot is compressed to WebP and archived in S3. All scores, designs, SEO audit data, and brand assets are persisted to the database. The user receives an instant shareable results page served via Vercel, with optional payment through Stripe to unlock the full design package — followed by a delivery confirmation email.
Vercel AWS S3 AWS RDS Stripe Resend
Platform Breakdown
Every Service, Explained
Ten cloud platforms working together — each chosen for reliability, speed, and best-in-class capability.
△
Hosting
Vercel
Web App & Serverless Functions
Hosts the entire PageRefresh web application and runs the analysis pipeline as serverless functions. Configured for up to 300 seconds of compute time per request — enough for even complex sites. Deployed to the US-Central region for low-latency access.
Serverless Functions 300s max duration Global CDN
🗃
Database
AWS RDS — PostgreSQL
Primary Data Store
Stores every analysis result: scores across 8 dimensions, SEO audits, extracted brand assets, all three generated HTML layouts, user contact data, Stripe payment status, and the industry benchmark dataset used for competitive scoring.
PostgreSQL Managed via Prisma ORM us-east-2
🌂
File Storage
AWS S3
Asset & Screenshot Storage
Stores all binary assets: compressed WebP screenshots of the analyzed site, extracted brand images, logos, hero photos, and team photos. Files are served with immutable long-term cache headers for fast repeat access.
Bucket: pagerefresh-assets Immutable cache Pre-signed URLs
🤖
AI — Primary
Anthropic Claude
Vision, Analysis & Design Generation
The core intelligence of PageRefresh. Claude Haiku handles fast tasks: screenshot analysis, industry detection, and logo identification. Claude Sonnet powers the heavy work: scoring against benchmarks, generating a creative brief, and producing three full HTML redesigns per analysis.
Claude Haiku 4.5 — Vision Claude Sonnet 4.6 — Design Prompt Caching
✨
AI — Secondary
OpenAI GPT-4o
Fallback AI Provider
Available as a secondary AI provider for layout generation and copy tasks. Ensures continuity and flexibility — the platform can route to GPT-4o when needed, preventing any single-provider dependency for critical AI outputs.
GPT-4o Fallback routing
📷
Screenshots
ScreenshotOne
Live Website Screenshots
Captures a pixel-perfect 1440×900 screenshot of any live website in real time. Automatically blocks ads and cookie consent banners so Claude Vision sees the real page content — not pop-ups. Non-blocking: the pipeline continues gracefully if a screenshot cannot be taken.
1440×900 desktop Ad & banner blocking PNG output
🔥
Web Scraping
Firecrawl
Bot-Protected Site Scraping
Handles websites protected by Cloudflare and other bot-detection systems that block standard HTTP requests. Firecrawl renders the page in a real browser environment and returns clean HTML — ensuring even challenging sites can be fully analyzed.
JavaScript rendering Cloudflare bypass Falls back to native fetch
⚡
Rate Limiting
Upstash Redis
Serverless Rate Limiting
Enforces per-IP rate limits using a serverless Redis instance. Prevents abuse and protects API costs without adding meaningful latency — checks complete in milliseconds before any analysis work begins.
Sliding window algorithm Serverless Redis Abuse prevention
💳
Payments
Stripe
Payment Processing
Handles all customer payments for design package purchases and layout installation. Stripe Checkout manages the full payment flow; signed webhooks confirm payment and instantly unlock design delivery — no manual intervention required.
Stripe Checkout Webhook verification PCI compliant
💌
Email
Resend
Transactional Email
Delivers all product emails: results notifications, payment confirmations, design delivery, and follow-up communications — all sent from the pagerefresh.ai domain for full brand consistency and high inbox deliverability.
